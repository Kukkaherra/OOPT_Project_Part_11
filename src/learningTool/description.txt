A compound pattern is a set of patterns that work together and they're useful when multiple patterns are being implemented in the design. There are many applications and use cases of compound patterns such as Model-View-Controller (MVC). MVC is a compound pattern which consists of View, Controller, and Model, and it is used to implement user interfaces. The Model is the back-end holding the data and logic. The View is the output of the system and it interacts with the Controller. The Controller is the logic of the system and it interacts with the Model and with the View. The Model consists of data.
